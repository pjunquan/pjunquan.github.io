# 俊权的个人网站 | Philip Pan's Personal Website

欢迎来到我的个人网站！这是一个展示我的专业背景、技能和AI教育实验室的交互式网站。

## 🌟 网站特色

- **多语言支持**: 简体中文、繁体中文、英文
- **响应式设计**: 适配各种设备屏幕
- **交互式体验**: 粒子动画背景、鼠标尾迹效果
- **AI教育实验室**: 展示AI在教育领域的应用

## 🚀 技术栈

- **前端**: HTML5, CSS3, JavaScript
- **框架**: Bootstrap 5.3.0
- **动画**: tsParticles, 自定义CSS动画
- **图标**: Font Awesome 6.4.0
- **字体**: Google Fonts (Inter)

## 📁 项目结构
